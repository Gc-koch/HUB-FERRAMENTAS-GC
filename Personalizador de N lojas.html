Como usar:

  1. Marque as caixas ao lado dos números que deseja incluir
  2. Use os botões "Selecionar Todos" ou "Remover Todos" para agilizar
  3. Utilize a caixa de busca para encontrar números específicos
  4. Use os botões AT-G, AT-M e AT-P para seleção automática por grupo
  5. O resultado formatado aparecerá automaticamente
  6. Clique em "Copiar Resultado" para usar em outro lugar
